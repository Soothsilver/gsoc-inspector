Detailed Description

Represents a simple member access expression.

Name resolution order: (After Stack frame) 1) slot name (local variable or parameter) - if state frame is provided 2) field name (declared in this class or static) / declared in predecessor, ... (if not a static method) 3) TypeName if type is predecessor of current instance then "restricted view" mode on current instance, otherwise view on Static field of given class 4) Error is reported

Name resolution order: (Deeper in the variable expression) 2) field name 3) Predecessor type name
