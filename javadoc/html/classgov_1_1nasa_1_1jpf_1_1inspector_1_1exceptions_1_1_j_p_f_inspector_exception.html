Detailed Description

Generic predecessor of all exceptions thrown by JPFInspectorBackEndInterface) implementors.

The subclass exceptions usually represent an error while executing a user command, for example a semantic error.

(All other exceptions thrown by JPFInspectorBackEndInterface are internal back-end errors.)
