Detailed Description

JLine SWING terminal.

The terminal is shown as scrollable text panel JEditorPane.

The terminal has to be created explicitly by constructor. Standard {

See also
Terminal::setupTerminal()} will not create this terminal.

The terminal creates own ConsoleReader. To get user input call

SwingTerminal.getConsoleReader().readline()

The getUserTextPrintStream() print stream can be used to print additional output to the console (just above the prompt line).
