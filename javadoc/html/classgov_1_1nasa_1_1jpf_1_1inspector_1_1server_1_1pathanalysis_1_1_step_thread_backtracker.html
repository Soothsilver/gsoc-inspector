Detailed Description

Backward step iterator for a given thread using the given TransitionThreadBacktracker.

Seamlessly hides transition boundaries and repetition of instructions at transition boundaries (top/bottom halves).

Step gov.nasa.jpf.inspector.server.pathanalysis.StepThreadBacktracker.backstep ( )

Performs a step back in the transition path using its TransitionThreadBacktracker, then returns the step that it back-stepped to.

The TransitionThreadBacktracker only considers transitions of the specified thread.

If the backstep fails (because there is no step we can backstep to), null is returned. This happens if our current step is the first step executed by the specified thread.

Returns
The step we backstepped to, or null if the backstep failed.

boolean gov.nasa.jpf.inspector.server.pathanalysis.StepThreadBacktracker.requestPreviousTransition ( )
private

Uses the TransitionThreadBacktracker to backtrack to the previous transition and sets this backtracker's current transition to that transition.

After this method terminates, the currentTransitionStepIndex will be set to the number of steps of the transition, i.e. a number greater by 1 than the topmost step. It is the responsibility of backstep() to decrement it. backstep() is the only method to call this one.

Returns
True if the previous transition was loaded; false if there is no previous transition.
