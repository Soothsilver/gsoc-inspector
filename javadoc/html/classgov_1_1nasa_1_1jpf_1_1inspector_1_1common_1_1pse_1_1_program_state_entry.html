Detailed Description

Abstract class that holds properties common to all program state entries.

A program state entry is created by the server in response to the "print" command and sent back. The print command then prints the program state entry as a string. That is the only use of this class.

This class is exposed as public API to custom hit conditions, therefore all of its methods (and the methods of its subclasses) should be well-documented.
