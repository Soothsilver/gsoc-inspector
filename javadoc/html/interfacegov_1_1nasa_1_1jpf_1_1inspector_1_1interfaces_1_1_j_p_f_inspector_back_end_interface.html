void gov.nasa.jpf.inspector.interfaces.JPFInspectorBackEndInterface.permitJpfToResumeAgain ( )

If JPF is currently paused and prevented from resuming, it will be woken up and allowed to resume.

If multiple preventJpfFromResuming() calls were made, all of them must be undone using this mehod before JPF resumes.

Synchronized on the access lock.

Exceptions
IllegalStateExceptionWhen JPF was not being prevented from resuming.

boolean gov.nasa.jpf.inspector.interfaces.JPFInspectorBackEndInterface.preventJpfFromResuming ( )

If JPF is currently blocked, then it will be prevented from resuming until permitJpfToResumeAgain() is called.

If JPF is currently running, this method will simply return false.

If multiple calls to this method are made, then all of them must be undone using permitJpfToResumeAgain() before JPF can resume again.

Synchronized on the access lock.

Returns
True if JPF is currently stopped and will be prevented from resuming; false otherwise.
