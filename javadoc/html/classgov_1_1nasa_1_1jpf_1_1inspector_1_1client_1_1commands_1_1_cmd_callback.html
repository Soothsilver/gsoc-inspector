Detailed Description

Represents a hidden callback command that suspends execution of commands until a callback of the specified kind is received from the server.

This only works in record execution mode.

Important only for CB_STATE_CHANGE.

This field indicates which state we expect the server to switch to. If this is null, it means that we don't care which state the server switches to, only that it changes states.
