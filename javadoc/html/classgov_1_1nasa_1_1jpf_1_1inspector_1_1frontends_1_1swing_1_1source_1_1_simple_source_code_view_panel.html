Detailed Description

The source code view panel allows the user to view the source code of the file where the currently executed instruction resides.

This is an older implementation that uses a simple JEditorPane and could function as a fallback in case RSyntaxTextArea somehow doesn't work.

However, RSyntaxTextArea seems to work reliably so this panel could conceivably be deleted altogether.
