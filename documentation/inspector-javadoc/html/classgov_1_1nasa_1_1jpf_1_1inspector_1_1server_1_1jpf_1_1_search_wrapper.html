Detailed Description

Wrapper (decorator) for the search class to use generic Search in the Inspector.

Only limited functionality is supported.

You have to specify which search class wrap (which search to use). Wrapped search class should by specified in the search.classwrapped configuration entry.

It means you should use search.class=gov.nasa.jpf.inspector.server.jpf.SearchWrapper search.classwrapped=YOUR ORIGINAL SEARCH CLASS
