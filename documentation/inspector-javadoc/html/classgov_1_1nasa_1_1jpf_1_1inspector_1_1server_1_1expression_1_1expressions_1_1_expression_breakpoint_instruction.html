Detailed Description

Used for internal Inspector purposes to implement backward single instruction steps.

Note: Should be used in the single JPF transition. Don't support backtracking. Note: Meaningful usage ... take back current transition (backtrack), restart the choice generator to previous state and reexecute the same choice again Note: Inspector has to call evaluateExpression(InspectorState) after each executed instruction, no optimalization permitted.
