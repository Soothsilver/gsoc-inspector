Detailed Description

This package contains "program state entries", classes that represent information that is passed from the client console to the Inspector server.

See the base class, gov.nasa.jpf.inspector.common.pse.ProgramStateEntry, for details.

This package also contains the gov.nasa.jpf.inspector.common.pse.PSEVisitor interface that the command "print" uses to print the entries out.
