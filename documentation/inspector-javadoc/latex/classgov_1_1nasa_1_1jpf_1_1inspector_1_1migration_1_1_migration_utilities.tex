\subsection{Detailed Description}
This class contains static functions that help with the migration of the J\+PF Inspector codebase from J\+P\+F6 to J\+P\+F8 during the 2016 update. 

Because the update may break things, some modifications that could have unintended consequences pass through this class so that when something breaks, we can fix it everywhere we broke it at the same time. When we are sure it works, this class will be removed and the functions\textquotesingle{} code inlined at the appropriate places.
