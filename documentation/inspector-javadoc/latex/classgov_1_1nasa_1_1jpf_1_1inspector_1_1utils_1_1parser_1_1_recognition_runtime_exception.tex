\subsection{Detailed Description}
Internal class. 

Is used to traverse errors between parser and parser facade. The A\+N\+T\+L\+R-\/generated parsers throw a Recognition\+Exception when they encoutner a parse error and this class merely wraps that exception as a Runtime\+Exception (the original exception is not unchecked).
