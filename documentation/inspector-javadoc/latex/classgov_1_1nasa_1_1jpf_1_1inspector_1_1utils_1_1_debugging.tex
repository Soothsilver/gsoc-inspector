Gets the logger that should be used inside J\+PF Inspector. 

It sets its default log level based on the \char`\"{}log.\+level\char`\"{} property in J\+PF configuration files. We must do this on our own because J\+PF logging utilities are not yet initialized by the time we need logging. This is because the J\+PF object itself is not yet created (it is only created in the shell when the Verify command is executed).

This also means that advanced logging configuration (specifically, the log.\+error and log.\+warning etc.) string arrays won\textquotesingle{}t work.
