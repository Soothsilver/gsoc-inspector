\subsection{Detailed Description}
Class used to handle callbacks. 

This class serializes callbacks in correct order and can postpone J\+PF stopped callbacks.

This class should handle all callbacks which should occur after J\+PF is stopped. (Prevent creation of separate threads for each callbacks and related non-\/determinism in callbacks order due to scheduling)
