\subsection{Detailed Description}
/$\ast$$\ast$ Shows cursor when focus gained. 

If \hyperlink{}{J\+Text\+Pane} is the read\+Only panel, then caret is not shown. Even if carret.\+set\+Visible() has been called then if panel is show again then the caret is hidden.

This listener repairs this behavior.
